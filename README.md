# kokosuki-fe

## Prerequisites

- Node.js 20
- pnpm 9

## Setup

```bash
$ git clone https://github.com/tsukuba-cojt-kokosuki/kokosuki-fe.git
$ cd kokosuki-fe
$ pnpm install
$ pnpm dev
```
